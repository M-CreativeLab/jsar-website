# Runtime

During runtime, JSAR relies primarily on Node.js, Babylon.js, and its own implementation of web-standard APIs:

Below are the version details of the dependencies for JSAR runtime:

| Dependency | Version |
| --- | --- |
| Node.js | v18.12.1 |
| Babylon.js | v6.8.0 |

> The above information will be updated with each JSAR version release.
