# Graphics backends

This specifies the supported backends that JSAR can run on.

| Backend  | OS      | Status    |
|----------|---------|-----------|
| OpenGLES | Android | Ok        |
| OpenGL   | macOS   | Partial   |
| Vulkan   | Android | Not yet   |
| Metal    | macOS   | Not yet   |
| D3D11    | Windows | Not yet   |
| D3D12    | Windows | Not yet   |
