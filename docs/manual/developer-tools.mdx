# Developer Tools

This chapter explains how to use the JSAR DevTools for Visual Studio Code to develop spatial apps.

## Installation

First, you can install the extension in the following ways.

Requirements:

- Install Visual Studio Code version 1.80.0 or above.

### Install from Marketplace(Recommended)

Search "JSAR DevTools" at [Visual Studio Marketplace](https://marketplace.visualstudio.com/) or directly visit [RokidMCreativeLab.vscode-jsar-devtools](https://marketplace.visualstudio.com/items?itemName=RokidMCreativeLab.vscode-jsar-devtools) to download and install.

### Install via `.vsix`

> If you can't access to the Visual Studio Marketplace, you could try this way to download this extension directly and install it manually.

- First, download the latest `.vsix` installation package from this address: [vscode-jsar-devtools-latest.vsix](https://jsar.netlify.app/installer/vscode-jsar-devtools-latest.vsix)
- Then, open Visual Studio Code.
- Press <Typography.Text keyboard>Ctrl + Shift + P</Typography.Text>, type "Extensions: Install from VSIX..."
- Select the downloaded `.vsix` installation package.
- Click OK.
