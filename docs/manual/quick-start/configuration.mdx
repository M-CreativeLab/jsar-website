# Configure the Development Environment

![screenshot](vscode-extension-screenshot.png)

Configure the development environment for JSAR space components by installing JSAR DevTools for Visual Studio Code. There are two ways to install the extension:

- Install from the VS Code extension marketplace: [RokidMCreativeLab.vscode-jsar-devtools](https://marketplace.visualstudio.com/items?itemName=RokidMCreativeLab.vscode-jsar-devtools).
- Install from a VS Code extension package (`.vsix`).

## Install Visual Studio Code

> If you already have Visual Studio Code installed, you can skip this step.

Go to [https://code.visualstudio.com/](https://code.visualstudio.com/) to download and install Visual Studio Code.

> Ensure that your Visual Studio Code version is greater than or equal to 1.80.0.

## Install Node.js

> If you already have Node.js installed, you can skip this step.

Go to [https://nodejs.org/](https://nodejs.org/) to download and install Node.js.

After a successful installation, check the Node.js and npm versions using the following commands:

```bash
$ node -v
$ npm -v
```

> Ensure that your Node.js version is greater than or equal to 18.0.0, or choose the latest LTS version.

## Install JSAR DevTools

### Install from the Marketplace

Go to [JSAR DevTools Extension Page](https://marketplace.visualstudio.com/items?itemName=RokidMCreativeLab.vscode-jsar-devtools) to install the extension, or search for "JSAR DevTools" in the VS Code extension marketplace.

### Install from `.vsix`

- First, download the latest .vsix installation package from vscode-jsar-devtools-latest.vsix.
- Then, open Visual Studio Code.
- Press <Typography.Text keyboard>Ctrl + Shift + P</Typography.Text> to open the command palette, and type "Extensions: Install from VSIX..."
- Select the .vsix installation package you downloaded.
- Click "OK."
