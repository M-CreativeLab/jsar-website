# 预览

## 打开预览窗口

- 首先，打开任何 `*.xsml` 文件
- 并选中打开的标签（Tab）页，在右上角可以找到打开场景视图的按钮
- 点击按钮，就可以打开场景视图

![screenshot](howto-run-project.png)

## 交互功能

在预览窗口中，提供了多种交互方式用于帮助开发者更好地调试。

### 旋转物体

通过鼠标左键拖动，可以旋转物体。

### 缩放视角

通过鼠标滑轮，可以缩放视角。

> 注：最近/最远有最大值，如果操作没有变化可能是因为已经在最远或最近距离，此时可尝试通过相反的方向操作。

### 重置视角

通过左侧按钮，点击后可重置视角。

### 重新加载

通过右侧按钮，点击后可重新加载当前应用。

### 热更新

通过修改 XSML 代码或任何项目内的其他文件，都会触发场景的重新刷新。

> 注：未来会参考 VirtualDom 实现更精确的热更新，而不是每次都重新刷新页面。
