# 快速开始

要进行 JSAR 的快速入门，开发者可以有以下几个途径：

- 通过安装 JSAR Devtools for Visual Studio Code 插件，可以在 VSCode 中直接创建 JSAR 项目，进行开发调试。
- 通过申请 Rokid AR Studio 开发者套件，获取样机后可直接在样机上进行开发调试。

本章节，我们将着重介绍如何通过 VSCode 插件进行 JSAR 开发，直接在样机上开发的流程可参考调试章节的步骤。

- [配置环境](./quick-start/configuration)
- [创建项目](./quick-start/create-project)
- [运行项目](./quick-start/run-project)
- [发布项目](./quick-start/publish-project)
