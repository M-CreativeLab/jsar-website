# API Reference

| API | Version | Documentation |
| --- | --- | --- |
| **Node.js** | v18.12.1 | [Documentation](https://nodejs.org/dist/v18.12.1/docs/api/) |
| **Babylon.js** | v6.8.0 | [Documentation](https://doc.babylonjs.com/typedoc/modules/BABYLON) |
| **Web-standard** | - | [Documentation](runtime/web-apis) |
| **JSAR** | - | [Documentation](runtime/jsar-internal-apis) |
