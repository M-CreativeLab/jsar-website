# 快速开始

在进行空间小程序的开发，开发者可以有以下几个途径：

- 通过安装 JSAR Devtools for Visual Studio Code 拓展，可以在 VSCode 中直接创建项目，并基于 JSAR DevTools 拓展提供的功能进行调试。
- 通过申请 Rokid AR Studio 开发者套件，获取样机后可直接在样机上进行开发调试。

本节将着重介绍如何通过 VSCode 拓展进行空间小程序的开发。

- [创建项目](./quick-start/create-project)
- [运行](./quick-start/run-project)
- [打包](./quick-start/build-project)
