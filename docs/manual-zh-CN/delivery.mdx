# 发布与分发

JSAR 空间小程序支持多种分发方式，包括：

- 通过 Rokid AR 商城分发到 Rokid AR Studio 设备（推荐）
- 通过 Github、Npm 或其他自定义服务服务发布为传统 Web 服务，用户可通过以下方式访问：
  - 通过 JSAR 提供的传统浏览器在线访问
  - 通过 Rokid AR Studio 设备的 JSAR Canary 应用扫码访问

JSAR Canary 是由 JSAR 团队发布在 Rokid AR Studio 中一个独立的空间应用（意味着不在桌面空间），它提供最新的 JSAR 运行时的预览版功能，用户可以通过在应用内扫码访问任何一个 XSML 文档，并在空间中初始化。

> JSAR Canary 仅用于开发者预览，运行环境与桌面空间的小程序环境不同，如果需要将你的小程序发布到 Rokid 商城，请使用 Rokid AR Studio 中的桌面空间小程序进行预览和调试，并按照商城的正式流程进行提交与发布。
