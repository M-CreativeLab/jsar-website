# Rokid AR 商店

通过 Rokid AR 商店，您可以将空间小程序发布到系统的桌面空间供用户使用。

## 开发规范

- 包大小限制在10MB内，如果遇到模型过大，可通过 [gltf-transform](https://gltf-transform.dev/) 进行压缩。
- 在桌面空间中运行的空间小程序，会被缩放在一个 30cmx30cmx30cm 的立方体中，因此在开发时，需要注意模型的大小，以及模型内部的位置。
- 务必遵守所在国家和地区的法律法规，特别是与内容审查和分发有关的法律。不要创建任何违反法律的内容。
- 内容准则： 请确保所有小程序的内容是适当的，不包含以下类型的内容：
  - 暴力、恐怖、仇恨或歧视性内容。
  - 色情、性别不当或淫秽内容。
  - 谣言、虚假信息或欺诈性内容。
  - 侵犯他人隐私、知识产权或著作权的内容。
  - 恶意软件、病毒或恶意代码。
- 性能和优化： 确保小程序的性能良好，能够在设备上流畅运行。
- 版权和知识产权： 确保你拥有或有权使用小程序中包含的所有内容，包括3D模型、音频、图像等。尊重他人的知识产权，并避免侵权行为。
- 社交责任： 作为开发者，要对社会负有责任感。不要利用小程序传播虚假信息、仇恨、暴力或不良价值观。

## 发布

> 当前不支持开发者自己提交，需通过开发者大赛按照 [大赛规则](https://s.rokid.com/2tnLB1c) 将空间小程序打包后提交给 Rokid 工作人员后上架。
