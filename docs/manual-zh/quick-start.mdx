# 快速开始

要开始使用 JSAR，您需要一个支持 JSAR 的 XR 设备或应用程序。因为 JSAR 类似于 [Chromium][]，它无法独立运行，需要一个宿主应用来构建和运行。以下是支持的设备表：

| 设备 | 状态 |
| --- | --- |
| [Rokid AR Studio][] | 开发中 |
| [Rokid AR Lite][] | 开发中 |

为了帮助开发者快速上手 JSAR，我们还提供了一个使用 JSAR 来运行 Web 应用、图片和 3D 模型的应用程序 —— _Play WebXR_。您可以从[这里](https://foobar)下载以下平台的版本：

- [x] Rokid AR Studio  
- [x] Rokid AR Lite  
- [ ] Quest 4  
- [ ] Quest 3s  
- [ ] Pico Neo 3  

安装到支持的设备后，您可以在输入框中输入 HTML URL 以运行。

在本节中，我们将介绍如何使用 _Play WebXR_ 以及如何使用 Web 技术开发 3D Web 应用。如果您是 Web 开发者但不熟悉 3D 开发，建议从本节开始学习。

- [环境搭建](./quick-start/setup)  
- [编写 HTML](./quick-start/write-html)  
- [编写 XSML](./quick-start/write-xsml)  
- [运行](./quick-start/run)
