# 代码风格

> 本风格仅适用于 JSAR 项目本身，对于空间小程序的开发者来说，可自行选择你喜欢的代码风格。

## TypeScript

JSAR 项目遵循 Google TypeScript 风格指南，具体请参考 [TypeScript Style Guide](https://google.github.io/styleguide/tsguide.html)。

## Rust

JSAR 项目遵循 Rustfmt 风格指南，具体请参考 [Rustfmt Style Guide](https://doc.rust-lang.org/nightly/style-guide/)。

## C/C++

JSAR 项目遵循 Google C++ 风格指南，具体请参考 [C++ Style Guide](https://google.github.io/styleguide/cppguide.html)。

## C#

JSAR 项目遵循 Microsoft C# 风格指南，具体请参考 [C# Style Guide](https://docs.microsoft.com/en-us/dotnet/csharp/programming-guide/inside-a-program/coding-conventions)。

## ShaderLab

JSAR 项目遵循 Unity ShaderLab 风格指南，具体请参考 [ShaderLab Style Guide](https://docs.unity3d.com/Manual/StyleGuide-Shaders.html)。
