# API Reference

JSAR 使用了 Babylon.js、Web-standard 以及 Node.js 的 API，因此对于希望查询 API 的开发者来说，我们只提供对应的版本号或 API 文档链接，而针对 JSAR 自己提供的 API 可以参考 [JSAR APIs](runtime/jsar-internal-apis)。

| API | 版本号 | 文档 |
| --- | --- | --- |
| **Node.js** | v18.12.1 | [文档](https://nodejs.org/dist/v18.12.1/docs/api/) |
| **Babylon.js** | v6.8.0 | [文档](https://doc.babylonjs.com/typedoc/modules/BABYLON) |
| **Web-standard** | - | [文档](runtime/web-apis) |
| **JSAR** | - | [文档](runtime/jsar-internal-apis) |
