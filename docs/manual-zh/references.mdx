# 参考文档

在本章节中，我们将提供一系列参考资料，包括技术文档和规范，以帮助开发者更深入地了解高级功能。

## 运行时 API

JSAR 是一个 Web 运行时，提供了一组遵循 Web 平台标准的 API，例如 DOM、WebGL、WebXR 和 WebAssembly。此外，还提供了两组额外的 API：一组是部分 Node.js API，用于测试和调试；另一组是 JSAR 特有的运行时 API。

- [Web APIs](./references/web-apis)
- [JSAR APIs](./references/jsar-apis)

## 图形

以下文档将帮助您了解 WebGL 在 JSAR 中的工作原理以及 JSAR 的统一混合渲染架构支持的图形后端。您还可以找到 JSAR 支持的 WebGL 扩展的参考资料。

- [WebGL 和拓展](./references/webgl-and-extensions)
- [图形后端](./references/graphics-backends)

## WebXR 与输入源

JSAR 实现了 [WebXR Device API][]，用于运行 WebXR 应用程序。您可以在这里找到有关 WebXR Device API 和 JSAR 支持的输入源的参考资料。

- [WebXR 基础](./references/webxr-basis)
- [WebXR 输入源](./references/webxr-input-sources)

[WebXR Device API]: https://developer.mozilla.org/en-US/docs/Web/API/WebXR_Device_API

## 配置

提供了一组设备配置，您可以使用这些配置来调试和测试您的应用程序。

- [运行时配置](./references/runtime-configuration)

如果您是希望将 JSAR 集成到自己的应用程序或平台中的开发者，可以在这里找到嵌入式 API 和指南。目前，JSAR 提供了原生 C++ API 和 Unity C# API 的支持。

- [XSML](./references/xsml)
- [SpatialCSS](./references/spatial-css)
