# 功能

在 JSAR DevTools for Visual Studio Code，提供如下功能。

## 代码编写

Visual Studio Code 作为代码编辑器，天然就提供了代码编写的功能，这里不再赘述。

## 代码高亮

- 在拓展中支持 XSML 高亮
- 在拓展中支持 SCSS 高亮
- TypeScript 由对应的 TypeScript 拓展支持，无需额外支持

## 智能提示

- 在拓展中支持 XSML 智能提示
- 在拓展中支持 SpatialCSS 智能提示
- 在拓展中支持 TypeScript 以及 JSAR APIs 的智能提示已通过 TypeScript 实现

## 预览

- 可在拓展中打开并预览 XSML 文件
- 可对预览窗口中的物体进行旋转和缩放
- 可重新加载 XSML 文件
- 预览 XSML 时，可实现预览窗口的热更新

## 调试

- 可查看 XSML 文件的运行结果，包括物体的位置、旋转、缩放等信息
- 可查看 XSML 文件的运行日志
- 可通过拓展调试 WebXR 接口
- 可通过拓展进行 Rokid AR Studio 的真机调试

## 打包

- 可通过拓展内功能打包空间小程序
