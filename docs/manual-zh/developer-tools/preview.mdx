# 预览

## 打开预览窗口

预览应用分为两种方式：

1. 预览项目，即打开 `package.json` 中的 `main` 字段指向的 XSML 文件来预览项目
2. 预览 `*.xsml` 文件，即打开任意 XSML 文件来预览该文件

### 预览项目

- 首先，打开一个 `*.xsml` 文件
- 并选中打开的标签（Tab）页，在右上角可以找到打开场景视图的按钮
- 点击按钮，就可以打开场景视图

![screenshot](howto-run-project.png)

> 通过该方式打开的是当前项目中 `package.json` 中 `main` 字段指向的 XSML 文件。

### 预览 `*.xsml` 文件

有两种方式可以预览 `*.xsml` 文件（即不需要在 `package.json` 配置 `main`）

- 在 Explorer 中找到以 `.xsml` 结尾的文件，右键点击后，选择："JSAR: Open in Scene View"
- 在文件编辑器中，右键点击后，选择："JSAR: Open in Scene View"

## 交互功能

在预览窗口中，提供了多种交互方式用于帮助开发者更好地调试。

### 旋转物体

通过鼠标左键拖动，可以旋转物体。

### 缩放视角

通过鼠标滑轮，可以缩放视角。

> 注：最近/最远有最大值，如果操作没有变化可能是因为已经在最远或最近距离，此时可尝试通过相反的方向操作。

### 重置视角

通过左侧按钮，点击后可重置视角。

### 重新加载

通过右侧按钮，点击后可重新加载当前应用。

### 热更新

通过修改 XSML 代码或任何项目内的其他文件，都会触发场景的重新刷新。

> 注：未来会参考 VirtualDom 实现更精确的热更新，而不是每次都重新刷新页面。
