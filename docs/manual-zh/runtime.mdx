# 运行时

JSAR 在运行时主要依赖 Node.js、Babylon.js 以及自己实现的 Web-standard APIs：

![jsar-architecture](jsar-architecture.png)

以下是 JSAR 运行时依赖的版本信息：

| 依赖 | 版本 |
| --- | --- |
| Node.js | v18.12.1 |
| Babylon.js | v6.10.0 |

> 以上信息会随着 JSAR 的版本更新而更新。
