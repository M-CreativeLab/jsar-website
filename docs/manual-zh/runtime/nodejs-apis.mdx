# Node.js APIs

JSAR 运行时基于 Node.js，因此为了考虑到 Node.js 开发者的易用性，JSAR 也开放了一些 Node.js 的 API 供开发者使用。

## `assert`

用于 JSAR 内部编写测试用例时使用，开发者如何有测试需求也可使用该模块。

> 具体文档可以参考 [Node.js Assert 文档](https://nodejs.org/dist/v18.12.1/docs/api/assert.html)。

## `Buffer`

用于增强 TypedArray 和 ArrayBuffer 的功能，Node.js Buffer 提供了更多方便的数据处理方法，开发者可以按需使用。

> 具体文档可以参考 [Node.js Buffer 文档](https://nodejs.org/dist/v18.12.1/docs/api/buffer.html)。
